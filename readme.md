
# Popeyes

Le but de ce projet était de découvrir comment utiliser et exploiter l'outil très puissant qu'est docker.
## Technologies utilisé

 - [Docker](https://www.docker.com/get-started/)
## Deployment

Pour déployer le projet sur votre machine vous devez d'abord clone le repo ou les fichier du dossier. Ensuite vous assurez que docker desktop est bien installé et ouvert, enfin lancé en utilisant la commmande :

```bash
docker compose build
```
